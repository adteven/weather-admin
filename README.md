<h1>weather-admin</h1>

## 用法

### 安装依赖

pnpm install

### 安装一个包

pnpm add 包名

### 卸载一个包

pnpm remove 包名

我认为你应该先 `fork` 项目去开发，以便我更新时您可以同步拉取更新！！！
